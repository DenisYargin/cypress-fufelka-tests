# Cypress E2E тесты 


## 📖 Описание проекта

Проект содержит E2E тесты, реализованные с использованием Cypress и TypeScript. Тесты покрывают четыре основных сценария:
- **Сценарий A**: Регистрация, получение бонуса и сохранение сессии
- **Сценарий B**: Создание клуба и навигация по вкладкам
- **Сценарий C**: Редактирование профиля и смена никнейма
- **Сценарий D**: Гостевой вход и привязка email (бонус)

## 🛠 Технологии

- [Cypress](https://www.cypress.io/) v15.11.0 — фреймворк для E2E тестирования
- TypeScript — типизация кода
- dotenv — управление переменными окружениями

## 🚀 Установка и запуск
1. Клонировать репозиторий:
git clone https://github.com/DenisYargin/cypress-fufelka-tests.git
cd cypress-fufelka-tests

2.Установить зависимости: npm install

3.Создать файл .env в корне проекта (не выложил, скину отдельно)

Запуск через ui: npx cypress open

Запуск через консоль: npx cypress run
